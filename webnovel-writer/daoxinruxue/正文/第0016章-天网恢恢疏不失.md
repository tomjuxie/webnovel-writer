# 第0016章-天网恢恢疏不失

午后，青灰色的云层压得很低。

太学府前的广场上，风声凄紧，吹得论道台四周悬挂的斑驳铁马发出叮叮当当的杂乱声响。台下早已被黑压压的人群围得水泄不通，除了帝都的学子与百姓，更有数位身着法家漆黑法袍与名家白衣的高阶修行者站立在暗角，一道道沉举而深邃的目光皆锁在石阶之下。

“嘎吱，嘎吱。”

一阵极其沉稳、缓慢的木轮转动声在寂静的广场上响起。

众人自发地向两侧退去，让开了一条直通论道台的黄土大道。只见墨玄机身着一身利落的黑狐劲装，神色清冷沉静，双手推着轮椅的边缘，不急不缓地超前走着。苏木提着药箱跟在后方，脚步扎实。

而轮椅上，顾道儒一袭胜雪的白衣，肩上搭着那件宽大的黑狐裘。他双手随意地交叠在腹前，脸色虽然在寒风中略显苍白，但那一双眼眸却亮如寒星，脊梁挺得笔直，透着一股近乎神圣的文人傲骨。

顾道儒坐在木质轮椅上，被墨玄机推着，缓缓停在了大周学宫百家论道台的青石台阶前。在众人近乎狂热的注视下，他扶着木椅扶手吃力起步，随后在轮椅上端坐坐下，神态一片安详超然。

“顾道儒，你果然来了。”

一道冷酷、毫无情绪波动的声音从台上缓缓压落。

商九律站在论道台正中，身形枯槁却笔挺如松。他那一身漆黑的法袍在狂风中猎猎作响，腰间悬着的青铜法尺散发着金属特有的冰冷寒光。他那一双鹰隼般的眼睛死死地锁在顾道儒身上，右手缓缓按住了青铜法尺。

“生死百家擂，既然契约已成，大秦商九律，在此代天执法。”

话音未落，商九律一步迈出。

“轰！！”

伴随着他这一步的踏下，一股森冷、机械、容不得半点变通的庞大法理真气，从他体内排山倒海般轰然爆发！四境巅峰的法理威压铺天盖地而来，虚空中的温度在这一瞬间骤然跌至冰点。他的双眸深处，闪烁起黑铁般的法理光泽，右手指尖在虚空中疯狂划过，结出一枚枚重逾千钧的法条符文：

“法者，国之大防！凡越界乱纲、动摇社稷规矩者，当以国法铁律，锁之！！”

“哗啦啦！！”

只听见一阵密密麻麻、令人牙酸的铁器摩擦声自论道台四周的青石板下轰然响起。无数道漆黑如墨的法律铁索从砖缝中暴涌而出，在半空中凝聚为一张巨大无朋的“黑铁法网”！法网的每一个网格都由冰冷严苛的刑律条文所化，带着大秦帝国那无情吞噬一切的恐怖国家气运，彻底将整座论道台和顾道儒的心神，封锁在其中。那黑铁法索上散发出的“罪”与“罚”之概念威压，几乎要让台下不少心神不坚的学子当场跪倒。

黑网沉沉压落，黑铁绞索在轮椅的四周发出刺耳的轰鸣，距离顾道儒单薄的身躯，已不过三尺之遥。

风暴正中心，顾道儒端坐在木轮椅上，肩上的黑狐裘被狂风吹得疯狂抖动。

但在那沉重如山的黑铁法网压迫下，他甚至连眼睫毛都没有颤抖一下。他只是安静地看着那密密麻麻、以惩罚和监禁为唯一的绝对教条的冰冷铁网。

顾道儒指尖轻轻抚过脑海泥丸宫中那卷绽放着紫气的《道德经》木牍，清冷沙哑的声音悠然响起：

“法令滋彰，盗贼多有。”

嗡——！！

随着这至简却击中痛处的真言吐出，虚空震动。漫天压落的黑铁锁链剧烈震颤，表面的刑律符文像被什么力量从内部侵蚀，开始明灭不定，隐隐出现蛛网般的裂痕。

商九律瞳孔骤然一缩，心神气海猛地一颤，一股血气直冲颅顶。

但他没有退。他能在法家走到立身巅峰，绝非无脑之辈！他猛地咬破舌尖，一口精血狠狠喷在腰间的青铜法尺上。法尺嗡鸣作响，瞬间暴发出刺目的漆黑幽光，将满布裂痕的黑铁法网再次粘合收紧！

“法者，国之权衡！规矩不立，万民皆乱！区区道言，岂能动摇大秦百年法统！！”

商九律怒喝，双手变印，虚空中再次凝聚出十余条代表“杖刑”、“黥刑”的巨大荆棘铁索，裹挟着国家明法的铁血意志，再度朝顾道儒狂暴抽落！

面对更趋狂暴的锁链，顾道儒眼眸平静，嘴角露出一抹清冷的傲骨，口中诵读不停，儒门金芒在眼底一闪而逝：

“道之以政，齐之以刑，民免而无耻；道之以德，齐之以礼，有耻且格。”

浩然金光如水波般自他坐下散开，所过之处，那些抽落的荆棘铁索在金光冲刷下，速度暴跌，表面的尖刺寸寸融化。

“儒门仁义，皆是避实就虚的空谈！如何补乱世之饥馑？！若是无规无矩，天下何以得平！！”商九律双目猩红，法袍在四溢的真气中炸裂开来，七窍已隐隐渗出血丝，却硬是凭着一股“唯法独尊”的偏执道心，咬牙死死稳住了即将崩溃的法网。

台下太学府的老儒们见状，也纷纷摇头叹息。法儒之争延续千年，法家的铁律在乱世确实有着无法替代的稳固作用。

“商九律，你错了。”

顾道儒端坐于木质轮椅上，看着拼尽全身法力来维持“法度”的黑发男子，声音不高，却宛如黄钟大吕，震动云霄：

“法，非禁民之具，亦非复古之枷。”

“治世不一道，便国不必法古；不期修古，不法常可！”

商九律身体猛地一震，那枚高悬在他头顶、几乎要炸裂的法理青铜尺，在这一瞬间竟停止了震颤。

顾道儒指尖轻叩，泥丸宫中深埋的“法家书册”在这一刻轰然泛起滔天的暗金色辉芒。他直视商九律，一字一句，吐出法家开天辟地的至高真理：

“故大人寄形于天地而万物备，历心于山海而国家富。上无忿怒之毒，下无伏怨之患……长利积，大功立，名成于前，德垂于后！！”

轰隆隆——！！

天空上那沉闷的乌云，在这一瞬间被生生撕裂出一条百丈宽的紫金色裂口！

那倾泻而下的紫金大道之光，在论道台虚空中无声地汇聚、交织，化作了一张横亘百丈的紫金天网。

天网恢恢，此乃大道之本体。

道家观之，是自然无为、疏而不失的至虚天道；儒家观之，是德礼为经、仁义为纬的浩然规矩；而法家至圣观之，则是开天辟地、德垂于后的至公秩序。百家殊途，万法同归，皆是这恢弘天网在世间的不同侧影。

然而这一次，商九律那原本代表了严苛与监禁的“黑铁法网”，在触碰到这法家至高“德法合一”的宏大真理时，不再是崩溃，而是发出阵阵清脆的共鸣鸣响。

黑色铁网如融化的春雪，化为千百道温润的墨金色真气河流，主动且顺从地融入了那张博大浩瀚的紫金天网之中！

“这……名成于前，德垂于后……”

商九律呆立在原地。他看着那漫天消融、最终温顺归流的墨金色天网，看着自己双手中那正散发出前所未有圣洁金芒的青铜法尺。

法家数百年，在大秦只取了“以刑禁民”的末流偏执，却丢了老祖宗“德垂于后、顺应天时”的无上大道！

顾道儒，不是在用法理摧毁他。

他是在，替法家补全那断裂了千年的至高道统！

“噗通。”

商九律泪流满面，身躯剧烈颤抖，竟在这论道台正中，当着台下千百名帝都学子与列国密探的面，面朝那轮椅上的白衣青年，重重地长跪了下去。

“大秦法家商九律……叩谢顾先生指点法经天道！”

他惨然一笑，双手猛地用力。

“咔嚓！”

那柄本已恢复圣洁金光的青铜法尺，被他生生折断！他主动碎去了自己那修持了半生的“以刑禁民”的偏执气海。虽然境界暴跌，但他体内的根基却在法家完整道统的温养下，保住了一丝纯净至极的法理种子。

“今日，商某便回大秦，闭关重整我大秦法家道统。此恩此德，商某永世不忘！”

商九律再次朝顾道儒深作一揖，随后扶着断裂的膝骨，步履蹒跚却眼神清明地走下了论道台，在一片死寂中，凄然退场。

生死百家擂，第二战大捷！

然而，这一次，台下没有喝彩，唯有无尽的死寂与震撼。

主角赢了，却不是用武力将法家废为死狗，而是用法家至高经典，将整个大秦法家折服！大周朝堂上的保守权贵们面面相觑，脸色惨白——从今日起，大秦法家，不仅不能再追杀顾道儒，反而欠了他一个天大的人情！

……

深夜。

寒蝉观外，再次下起了凄冷的秋雨。檐角的水滴打在松针上，发出单调而沉闷的“嗒嗒”声。

顾道儒靠在小木屋的竹椅上，脸色苍白得没有一丝血色，眉宇间满是疲惫。强行调动法家至高真理，对他这具残破经脉的负担不可谓不重。

“吱呀——”

木门被轻轻推开。

墨玄机端着一碗艾草药泥走了进来，指尖在触碰到顾道儒冰凉的手心时，微微停顿了一下，眼底闪过一抹极其克制的柔和。

而在院落的古松下，凄冷风雨中，却不知何时多了一个佝偻的身影。

“沙沙，沙沙。”

那是一名穿着打满补丁的粗布短衣、手执一柄破烂竹帚的扫地老者。他一下一下地扫着石阶上的松针，动作极其缓慢，却仿佛能将满天的雨声都容纳进那扫地的律动中。

老者扫完石阶，拖着竹帚走到木屋门槛前，大大咧咧地往门槛上一坐，从腰间摸出一杆斑驳的旱烟袋，啪嗒啪嗒抽了两口。

顾道儒睁开双眼，看着这位在第一章指点过他“异水之龙”、随后彻底消失了十余章的神秘老人。

“前辈，许久不见。”顾道儒沙哑开口。

老者吐出一口烟圈，烟雾在风雨中变幻莫测，他那一双浑浊却仿佛历经沧桑的眼睛在顾道儒身上扫了扫，嘿然一笑：

“小子，今天在台上，用法家经典补全了法家的道统。不错，比老朽预想的，要聪明得多。”

“前辈到底是谁？”顾道儒直视他。

老人吧唧吧唧抽着烟，看着屋外如注的冷雨，浑浊的眼中闪过一丝难言的悲凉与沧桑：

“三十年前，老朽也是这论道台上的常客。直到那天……一位儒门亚圣在此陨落，血雨下了三天三夜，染红了整个学宫的青石板。老朽才明白，有些道，不能在台上只靠嘴讲，得在台下，用脚去走。”

他站起身，拍了拍屁股上的尘土，拖着那柄破破烂烂的竹帚再度走入雨幕中：

“你现在的‘知’勉强算够了。但若是不去‘行’，终究只是空中楼阁。等你哪天真的做到了‘知行合一’，老朽自会告诉你，我的名字。”

老者的身影无声地消融在极深的夜雨冷雾中。

顾道儒端坐于木椅上，看着那空无一人的山门，又看了看脑海中那两卷亮起温润光华的经典。

“知行合一吗……”

他喃喃自语，因疲惫而紧绷的脸颊，在温暖的油灯与袅袅艾香下，彻底平复了下来。
